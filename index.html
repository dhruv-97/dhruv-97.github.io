IPU-GO App + Analytics Dashboard

IPU-GO is a complete cloud-based college management system that automates the routine tasks for a teacher and enables communication between teacher and students.

It provides facility to manage Student details, Timetable, Attendance, Examination Schedule, and Notices & Circulars.

It sorts and manages the data according to different constraints like Aggregate percentage, Backlog status, and Placement eligibility check to reduce manual work for teachers and student.

This was the most challenging project I have ever done since we were not provided with the result database from the University directly. Instead I had to apply custom script to parse the result PDFs which were uploaded on University's website into JSON format so that it could be stored in our database. There were a lot of other complexities involved such as gaining the support of the professors to support this new system which made it even more worthwhile.

Latest Research on Cross-Age Face Recognition with Weighted Encodings

Facial Recognition marks the culmination of years of research in Artificial Intelligence and Deep Learning. Though a majority of research is going on, there is still a scope of improvement in the existing techniques. Our faces change with time when age increases, while the images in our dataset remain the same. We intend to study the accuracy of Residual Neural Network (RN) for the purpose of cross-age face recognition.

We introduce the concept of Normalized Weighted Euclidian Distance, a new algorithm that gives a dynamic importance to each index while comparing the facial encodings, which further improves the performance of RN. The performance is compared for cross- age reference coding (CARC), Amazon Web Services (AWS) Rekognition and other techniques on the various data set viz., cross-age celebrity dataset (CACD) and a verification subset CACD-VS. RN and AWS Rekognition achieved 98.40% and 99.44% accuracy, respectively on the CACD-VS dataset while 99.40% is the highest achieved accuracy as seen by us on this dataset.
